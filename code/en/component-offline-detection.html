Cómo funciona?

Sencillo. En esta página, desactive la conexión de datos / WiFi. Aparecerá un mensaje notificándole que ahora está desconectado. Además, el mensaje a continuación cambiará.

Página deshabilitada

Puede haber casos en los que desee eliminar por completo el acceso a una página si esa página está fuera de línea. También puede hacerlo mostrando un mensaje que solicite al usuario que se conecte.

Donde se puede usar?

Esta función está integrada de forma nativa en nuestro código. Todas las páginas detectarán si está en línea o fuera de línea y mostrarán este mensaje. Apague Internet en cualquier página y aparecerá el mensaje.

Sin conexión

Esta página requiere una conexión a Internet para funcionar. Conéctese, encienda su WiFi o datos móviles para habilitar esta página.
